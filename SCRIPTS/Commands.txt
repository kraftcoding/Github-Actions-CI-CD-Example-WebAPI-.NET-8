NOTE: The project must be created with containerization features

1. Add Docker Support to project in VS
2. Build the Dockerfile
3. Right click on project "Open in Terminal"

docker login
docker build -t kraftcoding/githubactionscicdexamplewebapinet:master .
docker images
docker push kraftcoding/githubactionscicdexamplewebapinet:master
docker pull kraftcoding/githubactionscicdexamplewebapinet:master

docker-compose up --build
